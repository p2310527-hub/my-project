# AI Stadium Assistant

AI-powered assistant to improve physical event experience.

Features:
- Navigation
- Crowd control
- Food finder
- Emergency help

Tech:
- HTML
- CSS
- JavaScript
- OpenAI API

Setup:
1. Open `script.js`.
2. Replace `YOUR_OPENAI_API_KEY` with your OpenAI API key.
3. Save and reload the page.

Warning: Do not commit your API key to version control.
